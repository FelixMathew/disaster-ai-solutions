## Overview
DisasterAI is a full-stack machine learning application designed to demonstrate real-time disaster detection using image and video inputs.  
The system processes input data through a preprocessing pipeline and classifies it into **SAFE** or **DAMAGE** categories.

---

## 🚀 Features
- 📷 Image-based disaster detection  
- 🎥 Video analysis using frame extraction  
- ⚡ Real-time prediction using ML model  
- 🌐 Full-stack architecture (Frontend + Backend)  
- 📊 Confidence score-based output  
- 📱 Mobile-accessible deployment  

---

## 🏗️ System Architecture


User → Frontend (React) → Backend (FastAPI) → ML Model → Prediction → Output


---

## 🧠 Methodology

| Stage | Description |
|------|------------|
| Data Input | Image / Video upload |
| Preprocessing | Resize, Normalize, Frame Extraction |
| Feature Extraction | Model-based automatic feature learning |
| Model Training | Supervised learning using labeled dataset |
| Prediction | SAFE / DAMAGE classification |
| Output | Confidence score + result |

## 🧪 Model Details

| Model | Purpose |
|------|--------|
| Logistic Regression | Baseline classification |
| SVM | High-dimensional data handling |
| Random Forest | Ensemble learning |
| MobileNetV2 (CNN) | Image-based classification (final model) |

---

## 📊 Workflow

1. User uploads image/video  
2. Backend processes input  
3. Preprocessing applied  
4. ML model predicts result  
5. Output displayed with confidence
